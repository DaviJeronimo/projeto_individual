SEJA BEM-VINDO AO TYPE ONE!

O Type One é um projeto que foi desenvolvido para apoiar pessoas com diabetes tipo 1 por meio de uma tecnologia simples, intuitiva e acessível. A proposta é oferecer uma plataforma que faz leituras das glicemias, gerando indicadores importantes, juntamente com dashboards claras para acompanhamento diario dos valores glicêmicos, tudo pensado para facilitar o entendimento e auxiliar no controle glicêmico.

O foco do Type One é transformar dados em informações úteis, trazendo visualizações objetivas, rápidas e modernas, ajudando os usuários a compreenderem padrões, tomarem decisões melhores e promoverem mais autonomia no cuidado com a saúde.
